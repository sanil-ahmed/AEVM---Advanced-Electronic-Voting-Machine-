# AEVM - Advanced Electronic Voting Machine

A multi-layered secure voting system that integrates biometric hardware with a software management platform.

## Features
* **Biometric Verification:** Fingerprint and face recognition concepts for voter identity fraud prevention.
* **Admin Override:** Secure system for emergency validation.
* **Real-time Results:** Instant data sync with the backend database.

## Technologies Used
* **Languages:** Python
* **Hardware:** ESP32, Fingerprint Sensors (AS608), Face Recognition modules.
* **Database:** MySQL
